# 竖屏口播视频生成工作流

> 基于 HyperFrames 的 **9:16 竖屏口播 + 动效面板** 成片流程。  
> 与居左横版、全屏横版为 **并列的第三套工作流**。  
> 参考样板工程：`C:\hf_demo\projects\nov26-vertical`  
> 主构建脚本：`scripts/build_vertical.py`

---

## 0. 竖屏布局决策（核心）

原片若是 **纯净竖版口播**（人物占满、无新闻 UI），用 **fullscreen 模式** 全屏铺满。

若原片 **不可用**（如 nov26：上半新闻嘉宾 + 中间德文字幕 + 左下角评论者），用 **pip 模式**：

```
┌──────────────── 1080 ────────────────┐
│         深色背景 + ambient           │
│    ┌─────────────────────────┐       │  y=72
│    │  裁切口播人物 900×680    │       │  中上居中，圆角卡片
│    └─────────────────────────┘       │  y=752
│  ─────── 动效面板 780px ─────────    │  y=760~1540
│  GSAP 数据 / 流程 / CTA              │
│  来源↙                               │
│                                      │
│      全宽居中字幕 + 底部渐变          │  底部 220px 安全区
└──────────────────────────── 1920 ────┘
```

| 模式 | 何时用 | 口播处理 |
|------|--------|----------|
| **fullscreen** | 裁切后人物高度 / 画幅 ≥ 72% | 全屏 `object-fit:cover` |
| **pip**（nov26） | 原片含新闻 UI、德文字幕等 | FFmpeg 裁评论者 → 中上卡片 |

---

## 1. 产出规格

| 项 | 要求 |
|---|---|
| 画幅 | **1080×1920（9:16）** |
| 帧率 | 30 fps |
| 渲染 | HyperFrames 真实渲染 |
| 口播 | pip：`crop=580:720:0:1180` → 900×680 中上卡片 |
| 动效 | **人物下方** 780px 面板（z-index 10） |
| 字幕 | 最底部居中，z-index 30，220px 安全区 |

## 2. 口播裁切（去掉新闻条）

```powershell
# build_vertical.py 自动执行；手动验证裁切：
ffmpeg -y -ss 8 -i "C:\hf_demo\input\nov26-edit.mp4" `
  -vf "crop=580:720:0:1180,scale=720:720" -frames:v 1 probe-face.jpg
```

裁切目标：**左下角评论者**（Traditional Chinese 口播），**不是**上半屏新闻嘉宾。

产出：
- `assets/nov26-face-vertical.mp4` — 无音频裁切视频
- `assets/nov26-vertical-audio.mp4` — 原片音轨

## 3. 动画与 QA

- 5 个 `vs-scene*.html` 均有独立 GSAP timeline + `window.__timelines`。
- **禁止** 静态 HTML；**禁止** 用全屏原片当底（会带入德文 UI）。
- 渲染后 **必须** 各场景结束抽帧目视。

| 场景 | 抽帧 | 检查 |
|------|------|------|
| vs-scene1-hook | 5.60s | 中上人物 + 下方 +6.4% / 无德文 |
| vs-scene2-grid | 11.70s | 三行卡片 + 来源 |
| vs-scene3-flow | 18.45s | 横向 SVG + chip |
| vs-scene4-stats | 22.75s | +6.4% + € 柱 |
| vs-scene5-cta | 24.95s | CTA |

## 4. 命令速查

```powershell
python C:\hf_demo\projects\nov26-vertical\scripts\build_vertical.py
Set-Location C:\hf_demo\projects\nov26-vertical
npx hyperframes render --quality standard --output renders\nov26-vertical-v3.mp4 --fps 30

# 抽帧 QA
$v = "renders\nov26-vertical-v3.mp4"
$out = "renders\qa-v3"
New-Item -ItemType Directory -Force -Path $out | Out-Null
@(@("s1",5.60),@("s2",11.70),@("s3",18.45),@("s4",22.75),@("s5",24.95)) | ForEach-Object {
  ffmpeg -y -ss $_[1] -i $v -frames:v 1 -q:v 2 "$out\$($_[0]).png"
}
Copy-Item renders\nov26-vertical-v3.mp4 "C:\Users\Administrator\Videos\" -Force
```

## 5. 关键常量（`build_vertical.py`）

```python
FACE_CROP = "580:720:0:1180"  # 评论者裁切
FACE_TOP, FACE_W, FACE_H = 72, 900, 680
ANIM_TOP, ANIM_H = 760, 780
CAP_SAFE = 220
LAYOUT_MODE = "pip"  # 可用区域不够 → pip；够则改 fullscreen
```
