# 版式变体与安全区

FrameCraft 现支持以下人物版式。`aspect_ratio` 决定画幅，`layout_variant` 决定人物是否全屏与落位。

## 当前可用版式

### 1. `fullscreen`

- 含义：人物全屏铺满。
- 适用：情绪冲击强、人物表演是主视觉中心。
- 安全规则：
  - 主要信息层默认压在左侧安全区。
  - 字幕位于底部字幕带，避免遮住嘴型。
  - 信息卡要有足够体积，但必须避开脸部；横版人物居中时优先使用左侧和角落结构块。

### 2. `speaker_right`

- 含义：横屏非全屏版，人物在右侧卡片区，左侧为信息安全区。
- 推荐画幅：`16:9`
- 适用：需要更多表格、解释卡、金句框的横版视频。
- 设计规则：
  - 人物在右侧，左侧是主要信息舞台；左侧空白面积安全前提下尽量不超过 30%。
  - 左侧动画块应明显大于普通贴纸，优先使用流程图、对照表、时间线或大反应卡。
  - 字幕仍然居中于全画幅底部，不随人物框右移。

### 3. `speaker_top`

- 含义：竖屏非全屏版，人物在上方卡片区，下方为信息与字幕区。
- 推荐画幅：`9:16`
- 适用：竖版口播、上方人物 + 下方信息密集解释。
- 设计规则：
  - 下方信息区可承载卡片、表格、流程节点，但必须给底部字幕留出清晰空间。
  - 字幕居中，不得贴边，不得与下方信息卡重叠。

### 4. `speaker_center`

- 含义：人物居中安全区版，适合方形或特殊构图。
- 推荐画幅：`1:1`
- 适用：需要更平衡舞台感的构图。
- 设计规则：
  - 信息块围绕人物两侧或底部错峰出现，避免堆在人物正脸。

## 用户常用输入映射

- `fullscreen` / `全屏` -> `fullscreen`
- `nonfullscreen` / `非全屏` / `safe_zone` -> 自动映射
  - `16:9` -> `speaker_right`
  - `9:16` -> `speaker_top`
  - `1:1` -> `speaker_center`

## 四个测试版式

用于回归测试的标准矩阵：

1. 横屏全屏：`aspect_ratio=16:9` + `layout_variant=fullscreen`
2. 横屏非全屏：`aspect_ratio=16:9` + `layout_variant=nonfullscreen`
3. 竖屏全屏：`aspect_ratio=9:16` + `layout_variant=fullscreen`
4. 竖屏非全屏：`aspect_ratio=9:16` + `layout_variant=nonfullscreen`
